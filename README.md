# Detail

Simple websie using HTML,CSS,JavaScript and some library for Plaimanas test

You can check the deployed project at: https://frontend-test-plaimanas.vercel.app

## Getting Start

1. **Download repository**
   - Download repository by select Code > Download ZIP, then extract file.
   - If using Git you can clone repository

2.**Open the Project**
  - Option A (Manual): open index.html file in repository.
  - Option B (Live Server): If you are using VS Code or other code editor, right-click index.html and select "Open with Live Server".

### Library
- TailwindCSS
- DaisyUI

### Core Functionality

- **Responsive**: TailwindCSS, DaisyUI
- **Animation**: CSS, TailwindCSS, JavaScript
